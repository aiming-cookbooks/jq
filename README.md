jq Cookbook
===========

This cookbook installs [jq](http://stedolan.github.io/jq/).

Requirements
------------

- Linux
- Mac OS X

Usage
-----

Just include `jq` in your node's `run_list`:

```json
{
  "name":"my_node",
  "run_list": [
    "recipe[jq]"
  ]
}
```
